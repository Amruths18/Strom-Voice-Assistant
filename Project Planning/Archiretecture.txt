Detailed Architecture Explanation
1. Audio Input Layer
Listens continuously through microphone
Detects wake word “Hey Strom”
Noise reduction + filtering

2. STT Layer (Speech to Text)
Converts user voice → tex
Works offline (Vosk) and online (Whisper API) 

3. NLP Engine
Classifies user Intent
Extracts keywords and entities
Applies context memory

4. Command Router
Maps intent to correct module
Ensures modular and clean execution

5. Core Modules
System Control Module
(shutdown, restart, lock, open apps)
Task Manager Module
(alarms, reminders, to-do tasks)
Messaging Module
(WhatsApp/email messages)
Conversation Module
(general Q&A, chatting)
Search Module
(weather, news, definitions)

6. Response Generator
Converts module output into natural text response

7. TTS Layer (Text to Speech)
Converts text → spoken voice using pyttsx3 (offline)

8. Continuous Loop
After responding, system listens again
Recognizes stop word “Stop Strom” to pause listening
